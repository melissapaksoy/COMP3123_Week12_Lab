# React Redux + JWT Demo

This is my small self-directed research project showing how React, Redux, and JWT (JSON Web Token) work together in a simple authentication flow.

The app includes:
- Login page
- Fake backend that returns a dummy JWT
- Redux store managing user state
- Protected dashboard visible only after login
- Logout functionality
- Basic demonstration of dispatch, actions, reducer, state & subscribe
